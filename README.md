# GameOfLife_DxLib
*Conway's Game of Life made with DxLib library*

## Compiling
#### 1.Get the DxLib first.  
*DX Library Copyright (C) 2001-2018 Takumi Yamada.*  
Please refer to http://dxlib.o.oo7.jp/index.html  
#### 2.open the .sln and build.
